How it works: Both devices open this page. One creates a room, the other joins with the same code. Files and messages transfer peer-to-peer via WebRTC — no server stores your data.

🔑 Room Setup

Create a new room or join with the same code on both devices.

📋 Quick Steps

1

Open Beam on both devices — phone and PC, any browser.

2

Enter the same room code on both and tap Join.

3

Go to the Transfer tab once both show Connected ✓

⚠️ Not connected — go to the Connect tab and join a room first.

🖥️ Install Beam as a PC App

Install Beam as a PWA — sits in your taskbar like a real app. Works on Windows, Mac, Linux.

1

Open Beam in Chrome or Edge on your PC.

2

Click the install icon in the address bar (screen with ➕), or Menu → Install Beam.

3

Launch in its own window and pin to taskbar.

⚡ Self-Host Locally

For fastest transfers, serve it on your local network. Both devices on same WiFi.

1

Save this page as beam.html

2

Run in terminal:

python3 -m http.server 8080
3

On phone: http://[PC-local-IP]:8080/beam.html

💡 Find your IP: ipconfig (Windows) or ifconfig (Mac/Linux)

🔗 Works Anywhere

WebRTC punches through NAT — works across networks too, not just local WiFi.
